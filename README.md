# Simple Website Project

This is a basic web project built using HTML, CSS, JavaScript, and PHP.

## Features
- Multi-page website
- Image gallery
- Contact form
- Basic PHP backend

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP

## How to Run
1. Place the project folder inside `htdocs`
2. Start Apache using XAMPP
3. Open in browser:
   http://localhost/project-folder-name/
